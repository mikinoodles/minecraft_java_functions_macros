### What does this function do? 何をする関数か？
This function shoots a random firework (10 patterns) in front of the player with random offset.  
この関数はプレイヤーの前で10種類のうちからランダムな花火を打ち上げます。

### How do I install it? どうやってインストールするの？
Create a new datapack within your world save and place both .mcfunction files inside your namespace's function folder. (Supplementary instructions on the way)  
セーブデータ内に新規データパックを作成し、指定のnamespace下のfunctionフォルダーに.mcfunctionファイルを両方とも格納してください。(追加手順準備中)

### How do I use function in-game once installed? インストール後、どうやってゲーム内で関数を使用するの？
Type `/function <my_namespace>:shoot_random_firework_nearby` in chat bar, where:
* `<my_namespace>` should be replaced with the actual namespace the function resides in.

チャットに`/function <my_namespace>:shoot_random_firework_nearby`と記入。ちなみに:
* `<my_namespace>` には実際に関数が在籍しているnamespaceの名前を代入
